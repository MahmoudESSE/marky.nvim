1. Marky.nvim                                          *marky.nvim-marky.nvim*

`marky.nvim` letâ€™s you put a pin on file that you want to return to later


FEATURES                                      *marky.nvim-marky.nvim-features*

- **Pin** a file and return to it in the same position
- **List** all your pinned files
- **Delete** your pin


REQUIREMENTS                              *marky.nvim-marky.nvim-requirements*

- Neovim >= 0.9.0


INSTALLATION                              *marky.nvim-marky.nvim-installation*

lazy.nvim <https://github.com/folke/lazy.nvim>

>lua
    {
        "MahmoudESSE/marky.nvim",
        lazy = false,
        opts = {},
        config = function(opts)
            require("marky").setup(opts)
        end
    }

USAGE                                            *marky.nvim-marky.nvim-usage*

- **pin**`require("marky").pin()` marks a file with a pin and adds it the list
- **list**`require("marky").list()` lists the pinned files in a floating
    window
- **delete**`require("marky").delete(number)` deletes a file based on its id
    in the list
